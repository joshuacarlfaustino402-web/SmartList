# SmartList

A modern Flutter application that allows users to create categorized shopping lists with real-time price estimation.

## Purpose

SmartList is designed to be a simple yet powerful tool for managing your shopping. It helps you stay organized by categorizing your lists and provides a clear, running total of the estimated cost, empowering you to stick to your budget.

## Features

*   **Categorized Lists:** Group your shopping lists by categories like "Groceries," "Home," or "Work" for better organization.
*   **Real-Time Price Tracking:** Get an instant total price for each list as you add or modify items.
*   **Clean & Modern UI:** A beautiful and intuitive user interface built with Material 3, featuring both light and dark modes.
*   **Local Data Persistence:** All your lists are saved securely on your device, available whenever you need them, even without an internet connection.
*   **Responsive Design:** The layout adapts smoothly to both mobile phones and web browsers.

## Getting Started

### Prerequisites

*   You must have the [Flutter SDK](https://flutter.dev/docs/get-started/install) installed on your system.

### Running the Application

1.  **Clone the repository:**
    ```sh
    git clone https://github.com/joshuacarlfaustino402-web/SmartList.git
    ```
2.  **Navigate to the project directory:**
    ```sh
    cd SmartList
    ```
3.  **Install the dependencies:**
    ```sh
    flutter pub get
    ```
4.  **Run the app:**
    ```sh
    flutter run
    ```

## Building the APK

To generate a release version of the Android application, run the following command:

```sh
flutter build apk
```

The final APK will be available in the `build/app/outputs/flutter-apk/app-release.apk` directory.
